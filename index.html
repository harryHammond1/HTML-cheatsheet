How to use Lists in HTML and style them with CSS

These tables shows you how to write code to display lists on your website.

Lists in HTML

Code Type of list code Description
<ol> Ordered List This is for a list with Numbers in order
<ul> Unordered List This for a list with no order
<li> List item This is for each item of your list

Lists in CSS

Property and Value Applied Style
ul {list-style-type: square;} Styles the bullet points
ol {list-style-type: roman} Styles the ordered list to roman numerals
ul { list-style-image: url('sqpurple.gif'); } Add image to the list style type
